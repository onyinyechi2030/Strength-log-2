# Strength Log

Installable mobile-first PWA for tracking daily squats and other strength training.

## Files
Upload all files in this folder to the root of your GitHub repository.

## GitHub Pages
Settings -> Pages -> Deploy from branch -> main -> /root.

If your phone still shows an old/small version after updating, uninstall the home-screen app and reinstall it from the GitHub Pages URL. The old service worker/cache can keep showing the prior build.
