# Strength Log

A mobile-first Progressive Web App for tracking daily squats and other strength training.

## Features

- Daily squat counter with quick-add buttons
- Custom set picker
- Repeat last set, undo, and clear today
- Daily, weekly, and monthly goals
- Other strength training log with exercise chips and notes
- Progress dashboard with streaks, totals, best day, consistency, and a 14-day chart
- History with edit support
- CSV export
- Local/offline storage using the browser
- Installable as a phone home-screen app through GitHub Pages

## Deploy on GitHub Pages

1. Upload all files in this folder to the root of your GitHub repository.
2. Go to Settings → Pages.
3. Under Build and deployment, choose Deploy from branch.
4. Select `main` and `/ (root)`.
5. Open the published URL on your phone and choose Install or Add to Home Screen.
